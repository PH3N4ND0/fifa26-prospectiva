Apéndice Metodológico: Modelo DMD-26

El sistema predictivo unificado DMD-26 procesa estimaciones bajo un enfoque estricto de inferencia bayesiana sustentado en la distribución Dixon-Coles Bivariada de Poisson.

1. Distribución Bivariada de Goles Proyectados

La tasa de intensidad goleadora esperada para el equipo local (\(\lambda\)) y el visitante (\(\mu\)) integra algebraicamente la variable climatológica (\(\delta_{\text{clima}}\)):

$$\lambda = \alpha_i \beta_j \delta_{\text{clima}}$$ $$\mu = \alpha_j \beta_i$$

Donde \(\alpha\) es la tasa ofensiva, \(\beta\) la debilidad defensiva, y \(\delta_{\text{clima}}\) el modificador de fatiga biológica, configurado en 0.88 para equipos de posesión corta en climas superiores a 30°C.

2. Inferencia Bayesiana de Paso Único

Para evitar que eventos azarosos discretos descalibren el prior, el algoritmo se recalibra tras cada jornada con un filtro dinámico basado en la diferencia de goles esperados (xG):

$$\alpha_{i,\text{nuevo}} = \alpha_{i,\text{anterior}} + \eta(xG_{\text{home}} - x_{\text{esperado}})$$ $$\beta_{j,\text{nuevo}} = \beta_{j,\text{anterior}} + \eta(y_{\text{esperado}} - xG_{\text{away}})$$

Con una tasa de aprendizaje de \(\eta = 0.12\), el modelo se adapta de manera óptima al estado de forma real y el peaje neuromuscular de la competición.
